授权客户端示例

这是一个授权客户端的演示页面，展示如何集成授权验证功能。

使用说明

  1. 在输入框中输入16位授权码
  2. 点击"验证授权"按钮进行授权验证
  3. 验证成功后，客户端会自动管理token和刷新
  4. 点击"检查状态"可以查看当前授权状态
  5. 点击"销毁客户端"可以清理所有授权信息

集成代码示例

// 初始化授权客户端 const licenseClient = new LicenseClient({ apiUrl: 'https://your-domain.vercel.app/api', onTokenExpired: () => { console.log('授权已过期，请重新验证') // 处理授权过期逻辑 }, onValidationFailed: (error) => { console.log('授权验证失败:', error) // 处理验证失败逻辑 } }) // 验证授权 const result = await licenseClient.validateLicense('YOUR_LICENSE_KEY') if (result.success) { console.log('授权验证成功') // 继续执行软件功能 } else { console.log('授权验证失败:', result.error) // 显示授权错误信息 } // 检查授权状态 if (licenseClient.isAuthorized()) { console.log('当前已授权') // 执行需要授权的功能 } else { console.log('当前未授权') // 提示用户输入授权码 }
